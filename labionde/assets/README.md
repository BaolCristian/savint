# assets/

Cartella per la grafica e l'audio futuri del gioco:

- `sprites/` — la bionde (camminata nelle 4 direzioni), animali, personaggi
- `tiles/` — tessere della mappa (erba, coppi, acqua, campi) ispirate alle foto vere
- `audio/` — suoni: la ruota del mulino, il gallo, la pioggia, la moka

Il gioco oggi è un singolo `index.html` autonomo: quando questi asset
arriveranno, un piccolo step di build li incorporerà nel file (base64) per
mantenere la portabilità.
